CSS পরিচিতি

CSS (Cascading Style Sheets) হলো ওয়েব পেজের স্টাইল নির্ধারণের জন্য ব্যবহৃত একটি ভাষা। এটি HTML এলিমেন্টগুলিকে কীভাবে প্রদর্শন করা হবে তা বর্ণনা করে।

সাধারণ CSS উদাহরণ:
/* একটি paragraph স্টাইল করা */ p { color: navy; font-size: 16px; line-height: 1.6; margin-bottom: 1rem; } /* একটি বাটন স্টাইল করা */ .btn { display: inline-block; padding: 10px 20px; background-color: #4e54c8; color: white; border-radius: 5px; text-decoration: none; transition: all 0.3s ease; } .btn:hover { background-color: #8f94fb; transform: translateY(-2px); }

CSS সিলেক্টর

CSS সিলেক্টর ব্যবহার করে আপনি HTML এর নির্দিষ্ট এলিমেন্ট সিলেক্ট করে স্টাইল প্রয়োগ করতে পারেন।

সিলেক্টর উদাহরণ ব্যাখ্যা
এলিমেন্ট সিলেক্টর p সব <p> এলিমেন্ট সিলেক্ট করে
আইডি সিলেক্টর #header id="header" এলিমেন্ট সিলেক্ট করে
ক্লাস সিলেক্টর .btn class="btn" সব এলিমেন্ট সিলেক্ট করে
গোষ্ঠী সিলেক্টর h1, h2, h3 সব h1, h2, এবং h3 এলিমেন্ট সিলেক্ট করে
সন্তান সিলেক্টর div > p div এর সরাসরি সন্তান সব p এলিমেন্ট সিলেক্ট করে

CSS বক্স মডেল

CSS বক্স মডেল প্রতিটি HTML এলিমেন্টকে একটি বক্স হিসেবে বিবেচনা করে, যার মধ্যে content, padding, border, এবং margin থাকে।

বক্স মডেল ভিজুয়ালাইজেশন:

Content
Padding
Border
Margin
.box { width: 300px; /* Content width */ height: 200px; /* Content height */ padding: 20px; /* Space inside the border */ border: 5px solid #4e54c8; /* Border around padding */ margin: 30px; /* Space outside the border */ box-sizing: border-box; /* Includes padding and border in width/height */ }

CSS গ্রিড

CSS অ্যানিমেশন

CSS অ্যানিমেশন আপনাকে HTML এলিমেন্টগুলিকে JavaScript ছাড়াই অ্যানিমেট করতে দেয়।

অ্যানিমেশন উদাহরণ:

এই বক্সগুলি বিভিন্ন CSS অ্যানিমেশন প্রদর্শন করে।

/* কীফ্রেম অ্যানিমেশন সংজ্ঞায়িত করা */ @keyframes slideIn { from { opacity: 0; transform: translateX(-100px); } to { opacity: 1; transform: translateX(0); } } /* অ্যানিমেশন প্রয়োগ করা */ .animated-element { animation-name: slideIn; /* অ্যানিমেশনের নাম */ animation-duration: 2s; /* সময়কাল */ animation-timing-function: ease; /* সময়ক্রম ফাংশন */ animation-delay: 0.5s; /* বিলম্ব */ animation-iteration-count: 1; /* পুনরাবৃত্তি সংখ্যা */ animation-fill-mode: both; /* স্টাইল রক্ষণাবেক্ষণ */ }

CSS ট্রানজিশন

CSS ট্রানজিশন আপনাকে CSS প্রোপার্টি মানগুলির মধ্যে স্মুথ ট্রানজিশন তৈরি করতে দেয়।

ট্রানজিশন উদাহরণ:
ক্লিক করুন

বক্সটি ক্লিক করলে 45 ডিগ্রি ঘুরবে স্মুথ ট্রানজিশনের সাথে।

.element { width: 100px; height: 100px; background: blue; transition: all 0.5s ease; /* সকল প্রোপার্টির জন্য 0.5s ট্রানজিশন */ } .element:hover { width: 200px; background: red; transform: rotate(45deg); }

CSS ভেরিয়েবল (কাস্টম প্রোপার্টি)

CSS ভেরিয়েবল (কাস্টম প্রোপার্টি) আপনাকে রিসিউজেবল মান সংজ্ঞায়িত করতে দেয় যা পুরো ডকুমেন্ট জুড়ে ব্যবহার করা যাবে।

CSS ভেরিয়েবল ব্যবহার করে থিম তৈরি:
--primary
--secondary
--accent
:root { --primary-color: #4e54c8; --secondary-color: #8f94fb; --accent-color: #ff6b6b; --font-size: 16px; --spacing: 1rem; } .element { background-color: var(--primary-color); font-size: var(--font-size); margin-bottom: var(--spacing); } @media (max-width: 768px) { :root { --font-size: 14px; --spacing: 0.5rem; } }
